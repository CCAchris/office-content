
# DataLabel.Width Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the width, in points, of the object. Read-only.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **DataLabel** object.


### Return Value

Double
