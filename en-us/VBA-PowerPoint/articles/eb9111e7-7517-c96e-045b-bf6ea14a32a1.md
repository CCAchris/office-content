
# AxisTitle.Caption Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns or sets a  **String** value that represents the axis title text.

## Syntax

 _expression_. **Caption**

 _expression_A variable that represents an  ** [AxisTitle](8eddc95c-2353-43fa-c055-ee76de28009d.md)** object.
