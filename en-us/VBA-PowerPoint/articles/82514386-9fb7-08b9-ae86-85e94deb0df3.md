
# Chart.SetBackgroundPicture Method (PowerPoint)

 **Last modified:** July 28, 2015

Sets the background graphic for a chart.

## Syntax

 _expression_. **SetBackgroundPicture**( **_FileName_**)

 _expression_A variable that represents a  ** [Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|FileName|Required| **String**|The name of the file for the graphic.|
