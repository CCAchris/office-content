
# DisplayUnitLabel.Height Property (PowerPoint)

 **Last modified:** July 28, 2015

Returns the height, in points, of the object. Read-only.

## Syntax

 _expression_. **Height**

 _expression_A variable that represents a  **DisplayUnitLabel** object.


### Return Value

Double
