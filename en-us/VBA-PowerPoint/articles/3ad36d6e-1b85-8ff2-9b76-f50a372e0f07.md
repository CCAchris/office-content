
# CustomLayout.Name Property (PowerPoint)

 **Last modified:** July 28, 2015

Represents the display name of the specified custom layout. Read/write.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  **CustomLayout** object.


### Return Value

String
