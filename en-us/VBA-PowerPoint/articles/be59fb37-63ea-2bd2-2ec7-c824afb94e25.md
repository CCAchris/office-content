
# MediaFormat.Volume Property (PowerPoint)

 **Last modified:** July 28, 2015

Gets or sets the volume of the media. Read/write.

## Syntax

 _expression_. **Volume**

 _expression_An expression that returns a  **MediaFormat** object.


### Return Value

Single
