
# Application.AfterShapeSizeChange Event (PowerPoint)

 **Last modified:** July 28, 2015

Occurs after an object (shape, picture, text box, chart, SmartArt as examples) has been resized on the slide.

## Syntax

 _expression_. **AfterShapeSizeChange**(shp)

 _expression_A variable that represents a  **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|shp|Required| [Shape](1da93849-99e0-827e-ced3-c6cf7f8569f3.md)|The shape that was resized.|
|shp|Required|SHAPE||
