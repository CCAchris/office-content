
# NamedSlideShow.Delete Method (PowerPoint)

 **Last modified:** July 28, 2015

Deletes the specified  **NamedSlideShow** object.

## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  **NamedSlideShow** object.
