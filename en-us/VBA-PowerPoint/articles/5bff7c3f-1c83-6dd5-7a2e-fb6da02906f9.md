
# ColorSchemes.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **ColorScheme** object from the specified **ColorSchemes** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **ColorSchemes** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the single  **ColorScheme** object in the collection to be returned.|

### Return Value

ColorScheme
