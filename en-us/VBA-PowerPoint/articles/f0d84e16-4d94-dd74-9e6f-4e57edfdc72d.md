
# Presentations.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **Presentation** object from the specified **Presentations** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **Presentations** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the single  **Presentation** object in the collection to be returned.|

### Return Value

Presentation
