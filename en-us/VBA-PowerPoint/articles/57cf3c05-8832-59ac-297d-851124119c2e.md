
# Columns.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **Column** object from the specified **Columns** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **Columns** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the single  **Column** object in the collection to be returned.|

### Return Value

Column
