
# ProtectedViewWindow.Edit Method (PowerPoint)

 **Last modified:** July 28, 2015

Changes the password of the  **ProtectedViewWindow** object.

## Syntax

 _expression_. **Edit**( **_ModifyPassword_**)

 _expression_A variable that represents a  **ProtectedViewWindow** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|ModifyPassword|Optional| **String**|The new password to change to.|

### Return Value

Presentation
