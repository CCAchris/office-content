
# ShapeRange.Item Method (PowerPoint)

 **Last modified:** July 28, 2015

Returns a single  **Shape** object from the specified **ShapeRange** collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **ShapeRange** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the single  **Shape** object in the collection to be returned.|

### Return Value

Shape
