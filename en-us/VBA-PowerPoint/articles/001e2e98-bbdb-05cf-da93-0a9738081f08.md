
# Presentation.Convert2 Method (PowerPoint)

 **Last modified:** July 28, 2015

Converts a file to a different file type.

## Syntax

 _expression_. **Convert2**( **_FileName_**)

 _expression_A variable that represents a  **Presentation** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|FileName|Required| **String**|The name of the file to be converted.|
