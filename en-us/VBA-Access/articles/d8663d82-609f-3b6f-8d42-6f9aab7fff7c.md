
# Operation.Execute Method (Access)

 **Last modified:** July 28, 2015

Executes the specified operation.

## Syntax

 _expression_. **Execute**( **_bstrParameters_**)

 _expression_A variable that represents an  **Operation** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|bstrParameters|Optional| **String**|Specifies values for the parameters of the operation.|

### Return Value

Variant
