
# PivotLines.Count Property (Excel)

 **Last modified:** July 28, 2015

Returns the number of items in the  **PivotLines** collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **PivotLines** object.
