
# Chart.ProtectDrawingObjects Property (Excel)

 **Last modified:** July 28, 2015

 **True** if shapes are protected. To turn on shape protection, use the ** [Protect](5f46d721-021b-d615-12c6-78aab49df500.md)** method with theDrawingObjects argument set to **True**. Read-only  **Boolean**.

## Syntax

 _expression_. **ProtectDrawingObjects**

 _expression_A variable that represents a  **Chart** object.
