
# Chart.Calculate Event (Excel)

 **Last modified:** July 28, 2015

Occurs after the chart plots new or changed data, for the  **Chart** object.

## Syntax

 _expression_. **Calculate**

 _expression_A variable that represents a  **Chart** object.


### Return Value

nothing
