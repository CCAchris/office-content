
# FirstLetterException.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **FirstLetterException** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [FirstLetterException](e365a683-010a-a074-5563-f0cac1f410b2.md)** object.
