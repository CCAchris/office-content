
# Zooms.Item Method (Word)

 **Last modified:** July 28, 2015

Returns the specified  **Zoom** object.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [Zooms](1a4d5375-ad91-1eb9-77cb-4a6f8dcc3eb8.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **WdViewType**|The specified zoom type.|

### Return Value

Zoom
