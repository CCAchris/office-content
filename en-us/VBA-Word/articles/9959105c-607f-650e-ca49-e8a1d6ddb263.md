
# MailMergeDataFields.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **MailMergeDataFields** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [MailMergeDataFields](a660288d-1a2c-53ec-20d2-c52353be90c8.md)** collection.
