
# FormFields.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **FormFields** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [FormFields](a44a0f57-123b-cade-e306-ba6dc179b619.md)** collection.
