
# Tasks.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of tasks in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Tasks](ff521e20-8a25-f9f6-dccf-effea9debeb7.md)** collection.
