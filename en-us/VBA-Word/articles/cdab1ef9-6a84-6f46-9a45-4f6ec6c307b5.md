
# Series.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
