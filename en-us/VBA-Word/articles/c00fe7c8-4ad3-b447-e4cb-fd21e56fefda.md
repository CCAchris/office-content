
# BuildingBlocks.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **BuildingBlocks** object.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  ** [BuildingBlocks](be5bba4a-b06c-0074-20bd-bbeb40e03d1c.md)** object.
