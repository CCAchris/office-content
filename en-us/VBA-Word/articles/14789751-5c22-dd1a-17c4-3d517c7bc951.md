
# OMathMat.ColGap Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a  **Long** that represents the spacing between columns in a matrix. Read/write.

## Syntax

 _expression_. **ColGap**

 _expression_An expression that returns an  ** [OMathMat](40478b6e-18fe-b7b7-d0bc-def1349db56a.md)** object.
