
# Range.Document Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Document](8d83487a-2345-a036-a916-971c9db5b7fb.md)** object associated with the specified range. Read-only.

## Syntax

 _expression_. **Document**

 _expression_A variable that represents a  ** [Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.
