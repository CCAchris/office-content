
# Tasks.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Tasks** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Tasks](ff521e20-8a25-f9f6-dccf-effea9debeb7.md)** collection.
