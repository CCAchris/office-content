
# Sections.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Sections** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
