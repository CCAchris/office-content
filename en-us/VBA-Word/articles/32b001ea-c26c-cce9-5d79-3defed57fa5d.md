
# Page.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Page** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Page](3a3d480a-3876-515f-d13f-7ec23818245f.md)** object.
