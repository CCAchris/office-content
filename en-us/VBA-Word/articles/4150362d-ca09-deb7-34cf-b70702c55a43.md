
# Table.Select Method (Word)

 **Last modified:** July 28, 2015

Selects the specified table.

## Syntax

 _expression_. **Select**

 _expression_Required. A variable that represents a  ** [Table](996b58dd-ebc6-ee30-5bfe-c5e51a0f71d6.md)** object.


## Remarks

After using this method, use the  **Selection** object to work with the selected items. For more information, see [Working with the Selection Object](a1ef7e48-5a0f-d278-4b67-7b96f4e24052.md).
