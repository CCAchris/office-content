
# BuildingBlockEntries.Count Property (Word)

 **Last modified:** July 28, 2015

Returns the number of items in the  **BuildingBlockEntries** collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  ** [BuildingBlockEntries](9c5946e9-947d-7284-ab16-b570bf7f0ff3.md)** object.
