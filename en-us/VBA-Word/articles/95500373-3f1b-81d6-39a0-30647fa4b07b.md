
# Find.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Find** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Find](da822788-cad5-992a-a835-18cc574cc324.md)** object.
