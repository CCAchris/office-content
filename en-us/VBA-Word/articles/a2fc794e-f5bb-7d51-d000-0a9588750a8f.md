
# Windows.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Windows** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Windows](377b493b-e73c-0132-869c-3876c3beaef7.md)** collection.
