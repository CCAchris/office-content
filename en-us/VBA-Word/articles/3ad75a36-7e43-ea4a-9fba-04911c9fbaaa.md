
# CustomProperty.Value Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the value of a custom property. Read/write  **String**.

## Syntax

 _expression_. **Value**

 _expression_Required. A variable that represents a  ** [CustomProperty](1c4aa1ba-ad56-54d1-6e0d-2a82f7b9f4a9.md)** object.
