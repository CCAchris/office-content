
# Point.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.
