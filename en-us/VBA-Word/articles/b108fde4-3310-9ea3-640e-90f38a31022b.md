
# Editors.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a the  **Long** that represents the number of Editor objects in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents an  ** [Editors](acce718a-e3c1-deac-8b7f-fd8a5a9e47c6.md)** collection.
