
# Floor.Name Property (Word)

 **Last modified:** July 28, 2015

Returns the name of the object. Read-only  **String**.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  ** [Floor](01d277eb-501b-09e5-65b8-83506c76ac05.md)** object.
