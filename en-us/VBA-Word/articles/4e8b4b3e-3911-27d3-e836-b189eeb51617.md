
# Hyperlinks.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of hyperlinks in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [Hyperlinks](25801753-737f-9219-6a14-6531eb2ca699.md)** collection.
