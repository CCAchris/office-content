
# DataLabels.Shadow Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.

## Syntax

 _expression_. **Shadow**

 _expression_A variable that represents a  ** [DataLabels](a7676f18-b1f2-1e11-9489-863cb85c1669.md)** object.
