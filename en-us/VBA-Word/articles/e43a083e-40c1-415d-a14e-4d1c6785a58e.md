
# OMathArgs.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an  **OMath** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns an  ** [OMathArgs](5e4d542b-11c3-8cb8-be2a-5b990e777290.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMath
