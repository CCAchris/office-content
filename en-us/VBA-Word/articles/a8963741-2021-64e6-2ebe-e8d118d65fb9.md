
# TwoInitialCapsExceptions.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of exceptions in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [TwoInitialCapsExceptions](21af2d69-8d76-026d-2002-8d69b4ab8aef.md)** collection.
