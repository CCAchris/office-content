
# ChartCharacters.Insert Method (Word)

 **Last modified:** July 28, 2015

Inserts a string preceding the selected characters.

## Syntax

 _expression_. **Insert**( **_String_**)

 _expression_A variable that represents a  ** [ChartCharacters](cffe50a7-3fdc-75ad-2e32-081ba2310c1d.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|String|Required| **String**|The string to insert.|
