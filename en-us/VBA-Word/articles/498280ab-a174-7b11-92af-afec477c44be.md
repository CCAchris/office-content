
# Template.BuildingBlockEntries Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [BuildingBlockEntries](9c5946e9-947d-7284-ab16-b570bf7f0ff3.md)** collection that represents the collection of building block entries in a template. Read-only.

## Syntax

 _expression_. **BuildingBlockEntries**

 _expression_An expression that returns a  ** [Template](47d1d92d-bba9-3f2a-9c71-22ac43159bd3.md)** object.
