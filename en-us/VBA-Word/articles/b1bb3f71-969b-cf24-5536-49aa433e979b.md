
# Window.Hwnd Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that indicates the window handle of the specified window. Read-only.

## Syntax

 _expression_. **Hwnd**

 _expression_A variable that represents a  **Window** object.


## Property value

 **INT32**
