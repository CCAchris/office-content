
# Shapes.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Shapes** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Shapes](0907eed3-886e-8e73-0e5e-71f4b37ddd5b.md)** collection.
