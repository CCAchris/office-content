
# ContentControl.DateCalendarType Property (Word)

 **Last modified:** July 28, 2015

Returns or sets a  ** [WdCalendarType](437bf659-98fa-5a5b-bb74-7c956572db17.md)** constant that represents the calendar type for a calendar content control. Read/write.

## Syntax

 _expression_. **DateCalendarType**

 _expression_An expression that returns a  **ContentControl** object.
