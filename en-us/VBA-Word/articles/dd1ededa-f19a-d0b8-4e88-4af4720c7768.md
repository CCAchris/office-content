
# Chart.SideWall Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Walls](e98c7218-b944-12bb-caf9-daecee4b6c0c.md)** object that allows the user to individually format the side wall of a 3-D chart. Read-only.

## Syntax

 _expression_. **SideWall**

 _expression_A variable that represents a  ** [Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.
