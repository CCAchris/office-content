
# TextFrame.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Shape** object that represents the parent shape of the text frame.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [TextFrame](46f7e410-80d9-9fe9-2224-488b623f8592.md)** object.
