
# ListFormat.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ListFormat** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ListFormat](74773fd6-b713-34d4-b7be-f543c983008d.md)** object.
