
# Endnote.Delete Method (Word)

 **Last modified:** July 28, 2015

Deletes the specified endnote.

## Syntax

 _expression_. **Delete**

 _expression_Required. A variable that represents an  ** [Endnote](01f29be4-58e7-28f5-5fcb-dae50c33890e.md)** object.
