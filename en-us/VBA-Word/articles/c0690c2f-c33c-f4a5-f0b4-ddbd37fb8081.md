
# TablesOfAuthorities.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of tables of authorities in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [TablesOfAuthorities](c0fd88b1-b737-2811-ec4c-1fc274fc3e20.md)** collection.
