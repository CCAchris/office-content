
# Frameset.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Frameset** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Frameset](d76806db-c82f-f7b6-fb85-28a649de48a7.md)** object.
