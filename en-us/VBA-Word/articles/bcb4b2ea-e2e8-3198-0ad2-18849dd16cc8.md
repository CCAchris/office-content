
# BuildingBlock.Type Property (Word)

 **Last modified:** July 28, 2015

Returns a  **BuildingBlockType** object that represents the type for a building block. Read-only.

## Syntax

 _expression_. **Type**

 _expression_An expression that returns a  ** [BuildingBlock](2558b89f-8552-bb71-fa40-101cab2635ba.md)** object.
