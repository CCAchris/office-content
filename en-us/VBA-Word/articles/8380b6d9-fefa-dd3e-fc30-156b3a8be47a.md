
# BuildingBlocks.Item Method (Word)

 **Last modified:** July 28, 2015

Returns a  **BuildingBlock** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  ** [BuildingBlocks](be5bba4a-b06c-0074-20bd-bbeb40e03d1c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Specifies a  **String** or **Integer** that represents the name or ordinal position of the object in the collection.|

### Return Value

BuildingBlock
