
# Zoom.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Zoom** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Zoom](9a07fe91-fe6c-21f8-7022-1c56676b89ef.md)** object.
