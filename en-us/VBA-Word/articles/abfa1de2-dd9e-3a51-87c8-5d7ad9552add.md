
# MailMessage.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **MailMessage** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [MailMessage](d0109969-27f7-0180-c56d-5b49a3f0171b.md)** object.
