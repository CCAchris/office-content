
# FormFields.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of fields in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [FormFields](a44a0f57-123b-cade-e306-ba6dc179b619.md)** collection.
