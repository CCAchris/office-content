
# PlotArea.Name Property (Word)

 **Last modified:** July 28, 2015

Returns the name of the object. Read-only  **String**.

## Syntax

 _expression_. **Name**

 _expression_A variable that represents a  ** [PlotArea](72d30767-7cfc-3063-0b49-f9fbc129a52c.md)** object.
