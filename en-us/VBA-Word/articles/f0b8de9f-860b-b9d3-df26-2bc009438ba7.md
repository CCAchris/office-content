
# OMathBox.E Property (Word)

 **Last modified:** July 28, 2015

Returns an  **OMath** object that represents the base of the specified equation object. Read-only.

## Syntax

 _expression_. **E**

 _expression_An expression that returns an  **OMathBox** object.
