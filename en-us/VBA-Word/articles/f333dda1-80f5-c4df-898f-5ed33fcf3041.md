
# Task.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Task** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Task](8802fcd5-0947-2ea0-308a-376077633e34.md)** object.
