
# BuildingBlockEntries.Item Method (Word)

 **Last modified:** July 28, 2015

Returns a  **BuildingBlock** object that represents the specified item in the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  ** [BuildingBlockEntries](9c5946e9-947d-7284-ab16-b570bf7f0ff3.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Specifies a  **String** or **Integer** that represents the name or ordinal position of the object in the collection.|

### Return Value

BuildingBlock
