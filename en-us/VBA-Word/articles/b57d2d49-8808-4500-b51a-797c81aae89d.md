
# ListTemplates.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ListTemplates** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ListTemplates](5b5f3ed8-4522-f52e-5ae8-9df26a7da154.md)** collection.
