
# OMathMatRows.Count Property (Word)

 **Last modified:** July 28, 2015

Returns the number of items in the  **OMathMatRows** collection. Read-only **Long**.

## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  ** [OMathMatRows](beeb74aa-23ba-b9da-1f24-65c91fb3dc2c.md)** object.
