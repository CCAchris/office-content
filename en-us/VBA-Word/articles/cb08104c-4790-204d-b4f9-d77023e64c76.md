
# XMLSchemaReferences.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of XML schemas in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [XMLSchemaReferences](56bef973-805c-c77a-6d2a-54a39fbd1206.md)** collection.
