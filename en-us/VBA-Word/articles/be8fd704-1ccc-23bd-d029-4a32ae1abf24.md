
# Global.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Global** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Global](b91e7459-08d5-ea8c-42e0-f7b9bfd1a72c.md)** object.
