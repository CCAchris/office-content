
# ColorFormat.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **ColorFormat** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [ColorFormat](5f12793f-d847-ecf2-6cf6-39387f7f0b28.md)** object.
