
# AddIn.Name Property (Word)

 **Last modified:** July 28, 2015

Returns the name of an add-in. Read-only  **String**.

## Syntax

 _expression_. **Name**

 _expression_An expression that returns an  ** [AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** object.
