
# ListEntries.Count Property (Word)

 **Last modified:** July 28, 2015

Returns a  **Long** that represents the number of list entries in the collection. Read-only.

## Syntax

 _expression_. **Count**

 _expression_Required. A variable that represents a  ** [ListEntries](cfd3c706-5b69-338f-b104-e12526b89f47.md)** collection.
