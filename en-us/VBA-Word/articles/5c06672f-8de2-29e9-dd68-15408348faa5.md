
# Range.Footnotes Property (Word)

 **Last modified:** July 28, 2015

Returns a  ** [Footnotes](d46a0972-2784-4814-d547-30122a35cdc1.md)** collection that represents all the footnotes in a range. Read-only.

## Syntax

 _expression_. **Footnotes**

 _expression_A variable that represents a  ** [Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.


## Remarks

For information about returning a single member of a collection, see  [Returning an Object from a Collection](28f76384-f495-9640-a7c8-10ada3fac727.md).
