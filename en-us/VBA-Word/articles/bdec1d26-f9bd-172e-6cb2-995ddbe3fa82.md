
# Column.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **Column** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [Column](49d68571-2a57-6795-34b9-eb09aeb43043.md)** object.
