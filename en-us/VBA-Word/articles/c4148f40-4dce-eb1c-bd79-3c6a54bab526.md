
# RecentFiles.Parent Property (Word)

 **Last modified:** July 28, 2015

Returns an  **Object** that represents the parent object of the specified **RecentFiles** object.

## Syntax

 _expression_. **Parent**

 _expression_Required. A variable that represents a  ** [RecentFiles](c2d5e0b1-0d79-2fa7-c475-e5cace59ba1f.md)** collection.
