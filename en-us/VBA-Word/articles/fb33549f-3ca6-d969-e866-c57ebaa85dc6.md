
# ListTemplates.Item Method (Word)

 **Last modified:** July 28, 2015

Returns an individual  **ListTemplate** object in a collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_Required. A variable that represents a  ** [ListTemplates](5b5f3ed8-4522-f52e-5ae8-9df26a7da154.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

ListTemplate
