
# Application.Height Property (Word)

 **Last modified:** July 28, 2015

Returns or sets the height of the active document window. Read/write  **Long**.

## Syntax

 _expression_. **Height**

 _expression_A variable that represents an  ** [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.
