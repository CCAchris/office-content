
# RelatedShapePairEvent.Document Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Gets the  ** [Document](21640062-13a2-a2b2-7c61-7e707671207c.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_. **Document**

 _expression_A variable that represents a  ** [RelatedShapePairEvent](8a59ae03-ed45-21e3-73ad-8fdbe4c53299.md)** object.


### Return Value

 **Document**
