
# ContainerProperties.Application Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns the instance of Microsoft Visio that is associated with an object. Read-only.


## Syntax

 _expression_. **Application**

 _expression_A variable that represents a  ** [ContainerProperties](b94f758f-58f7-f1ef-c03b-761e26c11017.md)** object.


### Return Value

 ** [Application](5b3c8939-793f-116f-11b8-1d4170d95a63.md)**
