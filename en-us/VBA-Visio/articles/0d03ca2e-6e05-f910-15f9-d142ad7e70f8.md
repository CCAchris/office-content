
# VisTraceFlags Enumeration (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Flags to use with the  **Application.TraceFlags** property.


## Remarks

The value of the  **TraceFlags** property can be a combination of the following values.



|**Constant**|**Value**|**Description**|
|:-----|:-----|:-----|
| **visTraceAddonInvokes**|&amp;H4|Add-on invocations.|
| **visTraceAdvises**|&amp;H2|Outgoing advise calls.|
| **visTraceCallsToVBA**|&amp;H8|Microsoft Visual Basic for Applications (VBA) invocations.|
| **visTraceEvents**|&amp;H1|Event occurrences.|
