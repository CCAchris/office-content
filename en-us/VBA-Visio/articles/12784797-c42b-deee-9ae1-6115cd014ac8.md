
# Shape.Help Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Gets or sets the Help string for a shape. Read/write.


## Syntax

 _expression_. **Help**

 _expression_A variable that represents a  **Shape** object.


### Return Value

String


## Remarks

Using the  **Help** property is equivalent to entering a value in the **Help** box for a shape in the **Special** dialog box (in the **Shape Design** group on the [Developer](1bdc55f5-8fc7-7257-03d5-c049eceb29ff.md)tab, click  **Shape Name** ).
