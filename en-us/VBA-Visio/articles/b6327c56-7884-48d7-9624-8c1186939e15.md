
# ValidationRuleSets.Count Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns the number of  ** [ValidationRuleSet](cd2fc58a-5d7c-cf31-7aab-41bdeee9f105.md)** objects in the collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  ** [ValidationRuleSets](f08d7f04-13ec-8175-2aa6-94b0b67ee76b.md)** object.


### Return Value

 **Long**
