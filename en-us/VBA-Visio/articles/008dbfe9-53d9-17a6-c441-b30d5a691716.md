
# ContainerProperties.SetMargin Method (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Sets the gap between the container and member shapes to the specified size, in the specified units.


## Syntax

 _expression_. **SetMargin**( **_MarginUnits_**,  **_MarginSize_**)

 _expression_A variable that represents a  ** [ContainerProperties](b94f758f-58f7-f1ef-c03b-761e26c11017.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|MarginUnits|Required| ** [VisUnitCodes](fce91c1b-d5c2-6522-2446-0b8f6cacbc84.md)**|The units in which to measure the margin.|
|MarginSize|Required| **Double**|The size of the margin.|

### Return Value

 **Nothing**
