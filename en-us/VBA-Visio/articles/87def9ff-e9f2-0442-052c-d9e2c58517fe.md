
# Document.Type Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns the type of the  **Document** object. Read-only.


## Syntax

 _expression_. **Type**

 _expression_A variable that represents a  **Document** object.


### Return Value

VisDocumentTypes


## Remarks

Type value constants for  **Document** objects (the possible values that the **Type** property of a **Document** object returns) are declared by the Visio type library in ** [VisDocumentTypes](3872e4bd-cf79-66b3-b0bb-3ae0721a5435.md)**.
