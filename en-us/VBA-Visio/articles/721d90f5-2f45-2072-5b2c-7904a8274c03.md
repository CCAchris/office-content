
# Validation.ObjectType Property (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Returns  **visObjTypeValidation**, the type of this object. Read-only.


## Syntax

 _expression_. **ObjectType**

 _expression_A variable that represents a  ** [Validation](d59880de-ba16-eccf-fd94-f69da9a1efea.md)** object.


### Return Value

 **visObjTypeValidation**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the ** [VisObjectTypes](47d8b8f5-e402-7305-69d7-79d55d5c7f9e.md)** enumeration.
