
# Comment.Delete Method (Visio)

 **Last modified:** July 28, 2015

 _**Applies to:** Visio 2013 Preview_

Deletes the specified comment.


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  **Comment** object.


### Return value

 **VOID**
