
# CubeField.IsDate Property (Excel)

 **Last modified:** July 27, 2015

Returns  **True** if the CubeField is a date. Read-only **Boolean**.

## Syntax

 _expression_. **IsDate**

 _expression_A variable that represents a  **CubeField** object.
