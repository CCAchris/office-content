
# ChartArea.Width Property (Excel)

 **Last modified:** March 10, 2013

Returns or sets a  **Double** value that represents the width, in points, of the object.

## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **ChartArea** object.
