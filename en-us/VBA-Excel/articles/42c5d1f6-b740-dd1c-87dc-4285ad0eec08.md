
# DataFeedConnection.Creator Property (Excel)

 **Last modified:** March 10, 2013

 **In this article**
 [Version information](#sectionSection0)
 [Syntax](#sectionSection1)
 [Remarks](#sectionSection2)
 [Property value](#sectionSection3)


Returns a 32-bit integer that indicates the application in which the specified object was created.  **Long** Read-only


## Version information
<a name="sectionSection0"> </a>

Version Added: Excel 2013 


## Syntax
<a name="sectionSection1"> </a>

 _expression_. **Creator**

 _expression_A variable that represents a  [DataFeedConnection Object (Excel)](2ccb242b-28d5-3baf-78be-aa8f7478f4b6.md) object ( **Excel**).


## Remarks
<a name="sectionSection2"> </a>

Because the object was created in Microsoft Excel, this property returns the hexadecimal value, 5843454C, which represents the string "XCEL".


## Property value
<a name="sectionSection3"> </a>

 **XLCREATOR**
