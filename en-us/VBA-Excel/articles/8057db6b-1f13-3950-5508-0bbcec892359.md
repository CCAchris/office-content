
# Pages.Count Property (Excel)

 **Last modified:** March 10, 2013

Returns the number of objects in the collection. Read-only  **Long**.

## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **Pages** object.
