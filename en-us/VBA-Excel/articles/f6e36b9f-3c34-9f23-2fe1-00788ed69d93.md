
# Databar.Type Property (Excel)

 **Last modified:** March 10, 2013

 **In this article**
 [Version Information](#sectionSection0)
 [Syntax](#sectionSection1)
 [Remarks](#sectionSection2)


Returns one of the constants of the  ** [XlFormatConditionType](ae97c695-f56a-c9ee-91b0-dac413c93428.md)** enumeration, which specifies the type of conditional format. Read-only.


## Version Information
<a name="sectionSection0"> </a>

Version Added: Excel 2007 


## Syntax
<a name="sectionSection1"> </a>

 _expression_. **Type**

 _expression_A variable that represents a  **Databar** object.


## Remarks
<a name="sectionSection2"> </a>

This property will always return a  **Long** value of "4", which is equivalent to **xlDatabar**.
