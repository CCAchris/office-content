
# Top10.Delete Method (Excel)

 **Last modified:** March 10, 2013

Deletes the specified conditional formatting rule object.

## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Delete**

 _expression_A variable that represents a  **Top10** object.
