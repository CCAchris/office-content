
# WorksheetFunction.Parent Property (Excel)

 **Last modified:** March 10, 2013

Returns the parent object for the specified object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **WorksheetFunction** object.
