
# ChartTitle.Width Property (Excel)

 **Last modified:** March 10, 2013

Returns the width, in points, of the object. Read-only.

## Version Information

Version Added: Excel 2010 


## Syntax

 _expression_. **Width**

 _expression_A variable that represents a  **ChartTitle** object.


### Return Value

Double
