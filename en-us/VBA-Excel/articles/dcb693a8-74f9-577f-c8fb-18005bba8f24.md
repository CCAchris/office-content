
# Sort.Apply Method (Excel)

 **Last modified:** March 10, 2013

Sorts the range based on the currently applied sort states.

## Version Information

Version Added: Excel 2007 


## Syntax

 _expression_. **Apply**

 _expression_A variable that represents a  **Sort** object.
