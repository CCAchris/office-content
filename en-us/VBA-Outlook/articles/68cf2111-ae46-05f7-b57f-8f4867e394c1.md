
# AttachmentSelection.Item Method (Outlook)

 **Last modified:** July 28, 2015

Returns an  ** [Attachment](3e11582b-ac90-0948-bc37-506570bb287b.md)** object from the collection.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns an  **AttachmentSelection** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|Either the index number of the object, or the value used to match the default property of an object in the collection.|

### Return Value

An  **Attachment** object that represents the specified object.
