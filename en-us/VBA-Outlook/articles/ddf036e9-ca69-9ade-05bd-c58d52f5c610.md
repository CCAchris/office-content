
# IconView.Apply Method (Outlook)

 **Last modified:** July 28, 2015

Applies the  ** [IconView](dc2efa6c-4752-f713-f77e-378036f358dc.md)** object to the current view.

## Syntax

 _expression_. **Apply**

 _expression_A variable that represents an  **IconView** object.
