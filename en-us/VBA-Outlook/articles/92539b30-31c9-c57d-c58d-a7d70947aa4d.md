
# NotesModule.NavigationGroups Property (Outlook)

 **Last modified:** July 28, 2015

Returns a  ** [NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object that contains the set of navigation groups associated with the ** [NotesModule](cdbdde08-0773-a78d-3809-a3811975bcc1.md)** object. Read-only.

## Syntax

 _expression_. **NavigationGroups**

 _expression_A variable that represents a  **NotesModule** object.
