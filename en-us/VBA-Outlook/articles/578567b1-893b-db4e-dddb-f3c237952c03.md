
# MailItem.Body Property (Outlook)

 **Last modified:** July 28, 2015

Returns or sets a  **String** representing the clear-text body of the Outlook item. Read/write.

## Syntax

 _expression_. **Body**

 _expression_A variable that represents a  **MailItem** object.


## Remarks

The  ** [MailItem.BodyFormat](f635a0bc-20b7-206c-f558-a4ca2519670f.md)** property allows you to programmatically change the editor that is used for the body of an item.
