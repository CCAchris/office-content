
# TaskItem.BeforeAutoSave Event (Outlook)

 **Last modified:** July 28, 2015

Occurs before the item is automatically saved by Outlook.

## Syntax

 _expression_. **BeforeAutoSave**( **_Cancel_**, )

 _expression_A variable that represents a  **TaskItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Cancel|Required| **Boolean**|Set to  **True** to cancel the operation; otherwise, set to **False** to allow the ** [TaskItem](5df8cfa5-5460-a5a1-a130-ba5bca1a0091.md)** to be saved.|
