
# Pages.Add Method (Outlook)

 **Last modified:** July 28, 2015

Creates a new page in the  ** [Pages](ed4dd77e-b339-7f43-d036-c02daa69d5b8.md)** collection.

## Syntax

 _expression_. **Add** **_Name_**

 _expression_A variable that represents a  **Pages** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Name|Required| **String**| The name of the page.|

### Return Value

A  ** [Page](836941c3-c768-151a-65a5-41c71493033a.md)** object that represents the new page.


## Remarks

The  **Pages** collection is initially empty, and there is a limit of 5 customizable pages per collection.
