
# PlaySoundRuleAction.ActionType Property (Outlook)

 **Last modified:** July 28, 2015

Returns a constant,  **olRuleActionPlaySound**, from the  ** [OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.

## Syntax

 _expression_. **ActionType**

 _expression_A variable that represents a  **PlaySoundRuleAction** object.


## Remarks

 The sound playing rule action is supported only by rules for receiving messages. For more information, see [Specify Rule Actions](c5f83c81-0e01-38aa-5ec7-3932b4443e43.md).
