
# SharingItem.Save Method (Outlook)

 **Last modified:** July 28, 2015

Saves the  ** [SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** to the current folder or, if this is a new item, to the Outlook default folder for the item type.

## Syntax

 _expression_. **Save**

 _expression_A variable that represents a  **SharingItem** object.
