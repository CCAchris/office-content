
# ViewCombination.BottomView Property (Project)

 **Last modified:** July 28, 2015

Gets or sets a  ** [ViewSingle](41a36448-df16-3ad4-ec98-1dba0b3f8aef.md)** object representing the lower view of the combination view of the project. Read/write **ViewSingle**.

## Syntax

 _expression_. **BottomView**

 _expression_A variable that represents a  **ViewCombination** object.
