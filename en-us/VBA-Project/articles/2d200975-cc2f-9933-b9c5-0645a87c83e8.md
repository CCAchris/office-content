
# Reports.Parent Property (Project)
Gets the parent of the  **Reports** object. Read-only **Project**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Parent**

 _expression_A variable that represents a  **Reports** object.


## Property value

 **PROJECT**
