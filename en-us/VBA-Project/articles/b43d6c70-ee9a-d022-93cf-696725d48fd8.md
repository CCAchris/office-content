
# TaskDependencies.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **TaskDependency** object from the **TaskDependencies** collection. Read-only **TaskDependency**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **TaskDependencies** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the object to return.|
