
# ResourceGroups.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **Group** object from the **ResourceGroups** collection. Read-only **Group**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **ResourceGroups** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
