
# LookupTable.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **LookupTableEntry** object from the **LookupTable** collection object. Read-only **LookupTableEntry**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **LookupTable** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
