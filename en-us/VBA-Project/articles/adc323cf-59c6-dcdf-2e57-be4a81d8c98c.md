
# ShapeRange.Item Method (Project)
Gets an individual  **Shape** object in the shape range collection.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Item**(Index)

 _expression_A variable that represents a  **ShapeRange** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the shape to be returned. Can be an  **Integer** or **Long** for the ordinal position or a **String** for the name.|
|Index|Required|VARIANT||

### Return value

 **Shape**
