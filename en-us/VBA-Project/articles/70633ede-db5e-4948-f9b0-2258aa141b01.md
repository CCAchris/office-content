
# WorkWeekDays.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **WorkWeekDay** object from the **WorkWeekDays** collection. Read-only **WorkWeekDay**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **WorkWeekDays** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
