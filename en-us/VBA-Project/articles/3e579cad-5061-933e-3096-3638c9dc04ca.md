
# Exceptions.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **Exception** object from the **Exceptions** collection. Read-only **Exception**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents an  **Exceptions** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
