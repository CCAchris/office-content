
# ReportTemplates.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **ReportTemplate** object from a ** [ReportTemplates](01928892-d57c-8344-05db-d95008b4ba74.md)** collection. Read-only **ReportTemplate**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **ReportTemplates** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
