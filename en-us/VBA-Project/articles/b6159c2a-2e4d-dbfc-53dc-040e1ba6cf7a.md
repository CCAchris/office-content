
# Application.GanttBarTextDateFormat Method (Project)

 **Last modified:** July 28, 2015

Sets the date format for text around bars on the Gantt Chart.

## Syntax

 _expression_. **GanttBarTextDateFormat**( **_DateFormat_**)

 _expression_A variable that represents an  **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|DateFormat|Required| **Long**|A constant specifying the format of dates in the Gantt bar text. Can be one of the following  ** [PjDateFormat](d6bae7cd-4be0-b4eb-bbb1-5d82d7120bb2.md)** constants.|

### Return Value

 **Boolean**
