
# Application.CustomFieldDelete Method (Project)

 **Last modified:** July 28, 2015

Deletes the specified custom field.

## Syntax

 _expression_. **CustomFieldDelete**( **_Field_**)

 _expression_A variable that represents an  **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Field|Required| **Long**|A value representing a  ** [PjField](f0df0929-921c-1f33-ab42-192efdaeb64d.md)** constant.|

### Return Value

 **Boolean**
