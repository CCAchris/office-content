
# ResourceGroups.Copy Method (Project)

 **Last modified:** July 28, 2015

Makes a copy of a group definition for the  **ResourceGroups** collection and returns a reference to the ** [Group](e3756818-f051-1ae4-5402-0398e568ebfc.md)** object.

## Syntax

 _expression_. **Copy**( **_Name_**,  **_NewName_**)

 _expression_A variable that represents a  **ResourceGroups** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Name|Required| **String**|The name of the group to copy.|
|NewName|Required| **String**|The name of the new group.|

### Return Value

 **Group**
