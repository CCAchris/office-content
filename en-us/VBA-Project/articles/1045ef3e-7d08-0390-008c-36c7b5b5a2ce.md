
# Months.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **Month** object from the **Months** collection. Read-only **Month**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents a  **Months** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
