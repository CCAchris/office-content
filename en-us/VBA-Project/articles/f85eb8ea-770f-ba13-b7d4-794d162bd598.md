
# Shapes.SelectAll Method (Project)
Selects all shapes in the  **Shapes** collection.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **SelectAll**

 _expression_A variable that represents a  **Shapes** object.


### Return value

 **Nothing**
