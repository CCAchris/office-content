
# Shapes.Count Property (Project)
Gets the number of  **Shape** objects in the **Shapes** collection. Read-only **Long**.

 **Last modified:** July 28, 2015


## Syntax

 _expression_. **Count**

 _expression_A variable that represents a  **Shapes** object.


## Property value

 **INT**
