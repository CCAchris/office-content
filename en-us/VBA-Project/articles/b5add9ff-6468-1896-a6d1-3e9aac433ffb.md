
# Availabilities.Item Property (Project)

 **Last modified:** July 28, 2015

Gets a single  **Availability** object from the **Availabilities** collection. Read-only **Availability**.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_A variable that represents an  **Availabilities** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the object to return.|
