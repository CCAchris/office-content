
# BulletFormat2.Number Property (Office)

 **Last modified:** July 28, 2015

Gets the bullet number of a paragraph. Read-only.

## Syntax

 _expression_. **Number**

 _expression_An expression that returns a  **BulletFormat2** object.


### Return Value

Integer
