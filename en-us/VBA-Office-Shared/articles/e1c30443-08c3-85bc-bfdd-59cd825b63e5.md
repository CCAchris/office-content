
# MetaProperties.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **MetaProperty** object from the **MetaProperties** collection. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **MetaProperties** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Variant**|The name or index number of the  **MetaProperty** object to be returned.|

### Return Value

MetaProperty
