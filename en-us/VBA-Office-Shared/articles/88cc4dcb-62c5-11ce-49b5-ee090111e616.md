
# Font2.Italic Property (Office)

 **Last modified:** July 28, 2015

Gets or sets a value specifying whether the text for a selection is italic. Read/write.

## Syntax

 _expression_. **Italic**

 _expression_An expression that returns a  **Font2** object.


### Return Value

MsoTriState
