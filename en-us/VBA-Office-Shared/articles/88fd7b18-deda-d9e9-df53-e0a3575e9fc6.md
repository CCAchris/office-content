
# EffectParameter.Name Property (Office)

 **Last modified:** July 28, 2015

Retrieves the string name of the  **EffectParameter** parameter. Read-only

## Syntax

 _expression_. **Name**

 _expression_An expression that returns a  **EffectParameter** object.
