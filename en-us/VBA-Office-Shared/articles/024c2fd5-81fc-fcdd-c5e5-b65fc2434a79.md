
# CustomXMLPart.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **CustomXMLPart** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_An expression that returns a  **CustomXMLPart** object.


### Return Value

Object
