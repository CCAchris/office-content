
# TextRange2.InsertAfter Method (Office)

 **Last modified:** July 28, 2015

Inserts text to the right of the existing text in the  **TextRange2** object.

## Syntax

 _expression_. **InsertAfter**( **_NewText_**)

 _expression_An expression that returns a  **TextRange2** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|NewText|Optional| **String**|Contains the text to be inserted.|

### Return Value

TextRange2
