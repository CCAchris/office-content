
# WorkflowTemplate.Item Property (Office)

 **Last modified:** July 28, 2015

Gets a  **WorkflowTemplate** object from the **WorkflowTemplates** collection. Read-only.

## Syntax

 _expression_. **Item**( **_Index_**)

 _expression_An expression that returns a  **WorkflowTemplates** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
|Index|Required| **Long**|The index number of the  **WorkflowTemplate** object to be returned.|

### Return Value

WorkflowTemplate
