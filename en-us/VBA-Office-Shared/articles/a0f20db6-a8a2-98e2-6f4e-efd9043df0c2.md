
# CommandBarControl.Type Property (Office)

 **Last modified:** July 28, 2015

 **In this article**
 [](#sectionSection0)
 [Syntax](#sectionSection1)
 [Remarks](#sectionSection2)


Gets the type of  **CommandBarControl**. Read-only.


## 
<a name="sectionSection0"> </a>


 **Note**  The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, search Help for the keyword "ribbon."


## Syntax
<a name="sectionSection1"> </a>

 _expression_. **Type**

 _expression_Required. A variable that represents a  ** [CommandBarControl](b104ec00-beeb-a927-4b7b-108f4e3164f5.md)** object.


## Remarks
<a name="sectionSection2"> </a>

The value of the  **Type** property is available as a value in the ** [MsoControlType](62167ab6-edbe-4718-ff4a-0f1ff1059e8b.md)** enumeration.
