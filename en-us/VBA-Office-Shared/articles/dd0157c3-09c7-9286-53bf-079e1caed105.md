
# ThemeFonts.Parent Property (Office)

 **Last modified:** July 28, 2015

Gets the  **Parent** object for the **ThemeFonts** object. Read-only.

## Syntax

 _expression_. **Parent**

 _expression_An expression that returns a  **ThemeFonts** object.


### Return Value

Object
